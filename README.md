# Nabil's Portfolio

A showcase of my projects and skills built with React, Vite, and Tailwind CSS.

## Technologies Used

- React
- Vite
- Tailwind CSS 4.0

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Build

To build for production:

```
npm run build
```
